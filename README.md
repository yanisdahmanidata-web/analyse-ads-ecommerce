# Analyse de campagnes Meta Ads — E-commerce

## Objectif
Analyser les performances de 3 campagnes publicitaires Facebook 
et identifier les leviers d'optimisation pour maximiser le ROI.

## Résultats clés
- 84% du budget alloué sur la campagne la moins performante
- Profil client idéal identifié : femmes 30-34 ans (CPA 7 € vs 44 € en moyenne)
- Potentiel de +968% de conversions avec le même budget via réallocation

## Recommandations
- Augmenter le budget sur la campagne 916 (CPA 3,14 €)
- Cibler prioritairement les femmes 30-34 ans
- Réduire la campagne 1178 qui absorbe 84% du budget pour 3.95% de conversion
